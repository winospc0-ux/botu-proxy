# ملف الكوكيز لبوت يوتيوب
#
# الطريقة الأولى (سهلة - استخدم إضافة المتصفح):
# 1. ثبت إضافة "Get cookies.txt" على متصفح Chrome/Firefox
# 2. سجل دخول إلى youtube.com في المتصفح
# 3. استخدم الإضافة لتصدير الكوكيز واحفظ الملف كـ cookies.txt في نفس مجلد البوت
#
# الطريقة الثانية (يدوي):
# 1. افتح المتصفح وطوّر أدوات (F12)
# 2. اذهب إلى Tab Network، افتح أي فيديو يوتيوب
# 3. انسخ Request Headers > Cookie
# 4. الصق المحتوى في ملف cookies.txt
#
# ملاحظة: الكوكيز تسمح للبوت بالوصول للفيديوهات المقيدة بالعمر والمحتوى المحظور.
